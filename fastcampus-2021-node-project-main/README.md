# Fastcampus 2021 Node.js Project Example

강의의 백엔드 중간 프로젝트에 해당하는 저장소입니다.

터미널로 프로젝트 루트로 오신 후 다음을 실행해 주세요.
```bash
npm install
```

- `main` 브랜치엔 프로젝트의 뼈대만 있습니다.
- `.env` 파일을 만들어 프로젝트에서 사용되는 환경변수들을 설정해 주세요.

```bash
FB_APP_ID=
FB_CLIENT_SECRET=
NAVER_CLIENT_ID=
SERVER_SECRET=
MONGO_PASSWORD=
MONGO_CLUSTER=
MONGO_USER=
MONGO_DBNAME=
```

- `lecture` 브랜치엔 완성본이 들어 있습니다.
